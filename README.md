# Elemental Card Battle

**Elemental Card Battle** is a multiplayer card game for two players. Players fight using cards with different elements. The game is online and allows real-time battles.  

*Inspired by [CardJitsu](https://clubpenguin.fandom.com/wiki/Card-Jitsu) from Club Penguin.*

---

## Technologies

- **Frontend:** Next.js 13, TypeScript, Tailwind CSS, Zustand, Framer Motion, Three.js  
- **Backend:** Spring Boot, Java  
- **WebSocket:** STOMP + SockJS  
- **Database:** PostgreSQL  

---

## Current Features

- Lobby to create and join rooms  
- Real-time chat  
- Notifications when a player joins or leaves a room  
- Responsive interface
